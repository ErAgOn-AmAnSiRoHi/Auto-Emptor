## Tools and Libraries
* Gazebo
  > ```sudo apt install ros-$ROS_DISTRO-gazebo-ros-pkgs```
* OpenCV
  > ```sudo apt-get install libopencv-dev python3-opencv```
* PyZBar
  > ```pip install pyzbar```
* cv_bridge
  > ```sudo apt-get install ros-$ROS_DISTRO-vision-opencv```
